List of supported Atom text encodings

## Installation

`npm install @atxmtx/encodings -S`

## Usage

**Examples**:

```js
// JavaScript
import { encodings } from '@atxmtx/encodings';

console.log('List of supported encodings', encodings);
```

```coffee
# CoffeeScript
const { encodings } = require "@atxmtx/encodings"

console.log "List of supported encodings", encodings
