List of supported Atom text encodings

## Installation

`npm install @atxm/encodings -S`

## Usage

**Examples**:

```js
// JavaScript
import { encodings } from '@atxm/encodings';

console.log('List of supported encodings', encodings);
```

```coffee
# CoffeeScript
const { encodings } = require "@atxm/encodings"

console.log "List of supported encodings", encodings
```
